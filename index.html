Eshu Language Detector
Eshu is a language detector. It uses supervised leaning to guess more accurately the language of the text given by the user. It will take your sentence/name/text as input and tell you the language in which it was written. It is named after an African god in the Yoruba culture who knows all the languages on the surface of the earth and serve as a messenger between gods and humans. The software was developed with python IDLE using the thousands of book put available by the gutenberg project. For more info on the Gutenberg project consult https://www.gutenberg.org/ In this project we only used 20 books per language to train our learning algorithm. It supports French, English, Spanish and Germans.

Text Generator | Research project
I developed an application (in Python) that generates plausible human text, using Markov chains models and n-grams. The algorithm used was trained on books like Frankenstein, Sherlock Holmes accessed in txt files through datasets like the Gutenberg project. The version below used basic Markov chains to generate words. A problem here is that as the volume of data increases, it is more difficult to generate plausible text. A solution could be to add a second order markov chain or bigrams.
