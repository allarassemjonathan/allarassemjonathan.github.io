Eshu
Eshu is a language detector. It uses supervised leaning to guess more accurately the language of the text given by the user. It will take your sentence/name/text as input and tell you the language in which it was written. It is named after an African god in the Yoruba culture who knows all the languages on the surface of the earth and serve as a messenger between gods and humen. The software was developed with python IDLE using the thousands of book put available by the gutenberg project. For more info on the Gutenberg project consult https://www.gutenberg.org/ In this project we only used 20 books per language to train how learning algorithm.
